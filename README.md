♻️ EcoDetect AI - Waste Classification System

EcoDetect is an AI-powered web application that classifies waste into categories like cardboard, glass, metal, paper, plastic, and trash using image input. The goal is to promote smart waste management and environmental sustainability.

- Image Classification Model
- Input Size: 150x150
- Classes:
  - Cardboard

  - Metal
